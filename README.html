UN.plot.package

The goal of UN.plot.package is to generate any number of values from either uniform distribution or normal distribution. Plot the histogram for those values.

Installation

You can install the random.plot.package from GitHub with:

library(devtools)
devtools::install_github("tianyang1998/UN.plot.package")

Example

This is a basic example which shows you how to generate 100 values from normal distribution and uniform distribution. In addition, this example shows you the result histograms for these 100 values:

library(UN.plot.package)
## basic example code
rn <- random_normal(100)
ru <- random_unif(100)
